## Minetest monitor

Install libpcap:

```
brew install libpcap
```

or 

```
yum install libpcap-devel
```

or 

```
apt-get install libpcap-dev
```

Run monitor:

```
./gradlew minetest-monitor:run
```


Start a minetest server at localhost:30000.

Start a minetest client & connect to the local server.
